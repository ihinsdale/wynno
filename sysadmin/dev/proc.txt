1. Create EC2 instance with port 22, port 202, port 80, port 443 open to my IP
2. Change SSH port to 202
3. Run the Ansible site.yml playbook with the development inventory file:
   ansible-playbook -i development site.yml
4. SSH into the instance on Port 202
5. Open up a screen session
6. Launch in separate windows:
   - nodemon server.js
   - mongod --auth --dbpath ~/db/data
   - python server.py
   - sudo service redis-server start
   - sudo service nginx restart
